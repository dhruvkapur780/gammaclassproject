# Jaimin Shah – Personal Website

This folder contains a simple, single‑page professional website for **Jaimin Shah**.

## Structure

- `index.html` – Main website file.
- `assets/` – Folder for static assets.
  - `jaimin-shah-profile.png` – Circular‑cropped profile picture.

## How to use

1. Open this folder in your editor or file browser: `Jaimin-Shah`.
2. Place Jaimin's circular‑cropped headshot as `assets/jaimin-shah-profile.png`.
3. Open `index.html` in a browser (double‑click it or use a simple static server).
4. Edit text in `index.html` (summary, experience bullets, skills, contact email) to match Jaimin's LinkedIn profile.
