# Creative Studio Website

A professional single-page website for a design and development studio. Built with vanilla HTML, CSS, and JavaScript—no build step required.

## Features

- **Responsive design** — Works on desktop, tablet, and mobile
- **Refined animations** — Scroll-triggered fade-ins and subtle hover states
- **Professional theme** — Neutral palette with teal accent; Inter typography
- **Accessible** — Semantic HTML and ARIA where needed

## Quick Start

1. Open `index.html` in your browser, or
2. Run a local server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000` in your browser.

## Customization

- **Colors**: Edit CSS variables in `styles.css` (`:root` at the top)
- **Content**: Update copy in `index.html`
- **Contact**: Replace `hello@studio.example.com` and social links with your details
