A professional single-page personal website with placeholder content. Replace the bracketed text and links with your own information.

## Quick start

Open `index.html` in a browser, or run a local server:

```bash
# From this folder
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## What to replace

- **Hero**: Tagline, short bio, and call-to-action links
- **About**: Paragraphs about you and your photo (use an `<img>` in `.about-image`)
- **Experience**: Job titles, companies, dates, and descriptions
- **Projects**: Project names, descriptions, images, and Live/Code URLs
- **Skills**: Your actual skills (tech, tools, etc.)
- **Contact**: Email and social links (LinkedIn, GitHub, Twitter, etc.)
- **Footer**: Optional copyright line

All placeholders are marked with `[brackets]` in the HTML for easy search.
