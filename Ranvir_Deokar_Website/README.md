# Ranvir Deokar — Personal Website

A professional personal website with a dark, editorial design. Built with vanilla HTML, CSS, and JavaScript.

## Quick Start

Open `index.html` in your browser, or run a local server:

```bash
# Using Python
python -m http.server 8000

# Or using Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Customization

Replace the bracketed placeholders in `index.html` with your content:

- **Hero**: Tagline, CTA
- **About**: Photo (replace the avatar div with an `<img>`), bio paragraphs
- **Experience**: Job titles, companies, dates, descriptions
- **Skills**: Categories and skill lists
- **Projects**: Project names, descriptions, links, images
- **Contact**: Email, LinkedIn, GitHub, Twitter handles

## Design

- **Colors**: Dark background (#0f1419) with amber/gold accent (#d4a853)
- **Fonts**: Cormorant Garamond (headings), DM Sans (body)
- **Responsive**: Mobile-friendly with collapsible navigation
