# Riyaan Palle — Personal Website

A professional personal website with placeholder content. Replace bracketed content with your own information.

## Sections

- **Hero** — Name, title, tagline, and call-to-action buttons
- **About** — Bio and profile photo placeholder
- **Experience** — Timeline of work history
- **Skills** — Categorized skill lists
- **Projects** — Project cards with descriptions
- **Contact** — Email and social links

## Running Locally

Open `index.html` in a browser, or use a simple server:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Customization

1. Replace all `[placeholder]` text with your content
2. Add your photo in the About section (replace the placeholder div with an `<img>`)
3. Update the contact links (email, LinkedIn, GitHub, Twitter)
4. Add project images if desired
5. Link your résumé to the "Download Résumé" link
