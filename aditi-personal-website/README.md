# Aditi Das — Personal Website

A professional single-page personal website with placeholders ready for your content.

## Sections

- **Hero** — Name, tagline, and intro
- **About** — Photo placeholder and bio
- **Experience** — Timeline of work history
- **Projects** — Portfolio cards with screenshots
- **Skills** — Categorized skill tags
- **Contact** — Email and social links

## How to Run

1. Open `index.html` in a browser, or
2. Use a local server (e.g. `npx serve .` or `python -m http.server 8000`)

## Customization

Replace all `[placeholder]` text in `index.html` with your real content. Use `styles.css` to adjust colors (CSS variables at the top) and layout.
