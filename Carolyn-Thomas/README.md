# Carolyn Thomas – Personal Website

This is a simple, single-page personal website for **Carolyn Thomas**, designed in a clean **white and lavender** theme.

## Structure

- `index.html` – main page with sections for About, Experience, Education, Skills, and Contact.
- `styles.css` – typography, layout, and white/lavender styling (including circular profile photo).
- `images/carolyn-thomas-profile.png` – profile image used in the header.

Several content areas in `index.html` are clearly marked with comments where you can paste the real text from Carolyn's LinkedIn profile (headline, About, roles, and education).

## How to view locally

From a terminal:

```bash
cd /Users/sunehri/carolyn-thomas-site
open index.html
```

Or drag `index.html` into any modern browser.
