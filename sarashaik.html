About

Hi, I’m Sara.

I’m an Informatics major with a minor in Intelligent Systems Engineering at Indiana University, where I’m exploring how AI, systems design, and people-centered thinking come together. My work spans building AI decision-support tools like RideAgent, designing interactive learning experiences such as CyberCity, and thinking critically about how automation shapes our world through projects like The Automated World.

Structured customer support triage

Operations

Use this prompt to route incoming customer messages into consistent categories with suggested next actions. 

You are an assistant helping a customer support team triage incoming messages.

Goal:
- Classify each message
- Recommend the next best action
- Extract key details in a structured way

Instructions:
1. Read the message carefully.
2. Choose ONE primary category from: [Billing, Access, Bug, Product Question, Feature Request, Other].
3. Suggest the next action for a human agent in one sentence.
4. Extract any identifiers (user ID, email, account name, order number).

Output JSON with:
- category
- next_action
- key_details (object)

Message:
{{paste_customer_message_here}}

Selected Work

Projects & initiatives

RideAgent

Personal project · AI decision-support

Designed and built an AI-driven agent that compares multiple ride-hailing platforms to help users make more informed, cost-aware transportation decisions.

  • Integrated geocoding, provider abstraction, and fallback logic to handle inconsistent APIs and data gaps.
  • Structured the system to evaluate multiple scenarios, improving comparison accuracy and usability.
  • Treated the project as a modular decision-support tool that can grow with new data sources.

CyberCity

Course project · AI literacy game

Created an interactive Raspberry Pi–based game to help students aged 10–14 build intuition for how AI systems work through hands-on play.

  • Combined hardware, software, and tools like Fusion360, PrusaSlicer, and Adobe into a cohesive experience.
  • Translated abstract AI concepts into age-appropriate gameplay and visuals.
  • Balanced technical feasibility with accessibility and fun.

Inaam

Co-founder & Ops Lead · Ongoing

Co-founded and help run operations for Inaam, supporting construction efforts and ongoing orphanage operations while keeping the work financially sustainable.

  • Helped manage a six-figure operational budget with a focus on long-term impact.
  • Recognized with a Silver Award for sustained community service and project leadership.
  • Coordinated service-driven initiatives that connect resources to real community needs.
