# Elizabeth Vander Bie – Personal Site

This is a simple, fast, single-page personal website that you can customize for **Elizabeth Vander Bie**.

It is built with plain HTML, CSS, and a small amount of JavaScript so you can edit it easily without any framework or build tooling.

## Files

- `index.html` – main page structure and content.
- `styles.css` – layout, colors, typography, and responsive design.
- `script.js` – small enhancements (mobile navigation, theme toggle, dynamic year).

## Getting started

1. Open `index.html` in any web browser (double-click it from your file explorer).
2. Edit the text content directly in `index.html` to reflect your real biography, roles, and projects.
3. Update links in the **Contact** section (email, LinkedIn, other profiles).
4. Optionally tweak colors, spacing, or typography in `styles.css`.

## Customization tips

- Replace placeholder text (for example, role titles, company names, and project descriptions) with Elizabeth's real information.
- Swap the `EVB` avatar block in the hero section for a real photo if you’d like, by replacing it with an `<img>` tag and adjusting the styles.
- The contact form is front-end only; if you want it to send emails, connect it to a form provider or a backend endpoint.

## Dependencies

There are **no runtime dependencies**. This is a static site that can be hosted on any static hosting service (GitHub Pages, Netlify, Vercel, etc.) or served directly from a simple web server.
