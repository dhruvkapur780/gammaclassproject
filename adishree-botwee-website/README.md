# Sahiti Midithuri - Professional Website

A clean, professional single-page portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive design** — Works on desktop, tablet, and mobile
- **Smooth navigation** — Fixed header with smooth scroll to sections
- **Professional layout** — Hero, About, Expertise, Experience, Work, Testimonials, Contact
- **Placeholder content** — All sections include bracketed placeholders for easy customization

## Getting Started

1. Open `index.html` in a browser, or
2. Run a local server:
   ```bash
   # Python 3
   python -m http.server 8000

   # Node.js (npx)
   npx serve .
   ```
3. Visit `http://localhost:8000`

## Customization

Replace all `[placeholder]` text with real content:

- **Hero** — Tagline, bio, professional photo
- **About** — Full bio, stats (years, projects, clients), skills list
- **Expertise** — Service names and descriptions
- **Experience** — Job titles, companies, dates, descriptions
- **Work** — Project images, titles, categories, descriptions
- **Testimonials** — Client quotes and attributions
- **Contact** — Email, LinkedIn, location, form (connect to backend for submissions)
